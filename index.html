Tentang Saya

Biografi

Halo saya Boemi Pangkey seorang pengembang web pemula dengan pengalaman 1 bulan, saya suka pengembangan web di umur saya yang ke 14 tahun. Saya sangat tertarik dengan UI Design, AI, IOT, saya selalu siap untuk mempelajari hal baru.

Meskipun baru memulai perjalanan saya sebagai pengembang web, saya telah belajar banyak tentang HTML, CSS, dan JavaScript. Saya senang bereksperimen dengan desain antarmuka yang inovatif dan mencari cara baru untuk mengintegrasikan kecerdasan buatan serta internet of things dalam proyek-proyek sederhana. Saya percaya bahwa belajar adalah proses yang berkelanjutan, dan saya selalu mencari tantangan baru untuk meningkatkan kemampuan saya.

Umur: 14 tahun Pengalaman: 1 bulan Minat: UI Design, AI, IoT
